# Клининг Сервис Уфа (Ufa Cleaning Service)

Профессиональный лендинг для клининговой компании в Уфе.

## Особенности
- **Мобильная адаптивность**: Сайт отлично выглядит на телефонах и десктопах.
- **Современный дизайн**: Использован стек Tailwind CSS для стильного оформления.
- **Форма заявки**: Встроенная валидация и имитация отправки формы.
- **SEO-оптимизация**: Правильная семантика HTML5.

## Запуск

Этот проект не требует установки Node.js или сборки.

1. Скачайте все файлы.
2. Откройте файл `index.html` в любом современном браузере.

## Настройка

- Отредактируйте `index.html`, чтобы изменить цены, телефон и тексты.
- Изображения подгружаются с Unsplash (можно заменить на свои).

## Технологии
- HTML5
- CSS3 (Tailwind CSS CDN)
- JavaScript (ES6+)
